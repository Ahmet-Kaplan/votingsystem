• En el momento de recibir un voto el Control de Acceso comprueba que:
    • El Usuario no ha votado.
    • El Usuario proporciona un identificador de voto único.
    • El Usuario proporciona un identificador de solicitud de acceso único.
  • Si todo es correcto el sistema proporciona al Usuario un certificado con el que poder firmar de forma anónima el voto.
  • Una vez recibido el voto, si todo es correcto, el sistema devuelve un recibo firmado con:
    • El certificado de voto enviado al Usuario
    • El certificado del Centro de Control
    • El certificado del Control de Acceso
  • El protocolo del Sistema de Votación permite comprobar que el valor del voto del usuario es corréctamente contabilizado en los resultados finales porque tanto el Centro de Control como el Control de Acceso hacen público el recibo firmado en el momento en que responden al Usuario.
  • Tanto el Centro de Control como el Control de Acceso hacen públicos los identificadores (identificador de voto e identificador de solicitud de acceso) asociados al voto. De esa forma se puede comprobar el valor del voto asociado a un determinado identificador.
  • El usuario puede comprobar que hay un voto asociado al identificador proporcionado por él y si no está de acuerdo con el resultado puede anularlo.
  • Desde la página de información del evento se pueden encontrar las URLs de acceso al recuento en tiempo real de los votos.
